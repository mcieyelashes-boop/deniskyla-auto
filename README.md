# deniskyla.auto

**Agentic Dashboard** — CEO-led orchestration system with 7 specialized sub-agents.

## Architecture

```
CEO Agent (Orchestrator)
├── Website Developer
├── Market Research
├── Lead Gen
├── Email Campaign
├── Social Media
├── Content Creation
└── Content Scheduler
```

## Features
- CEO agent mengatur dan mendelegasikan ke sub-agents
- Orchestration presets (Product Launch, Growth Sprint, Content Blitz)
- Real-time progress & activity log per agent
- Powered by Claude API

## Stack
- React + Vite
- Claude API (Anthropic)
- Vercel (deployment)

## Setup
```bash
npm install
npm run dev
```
